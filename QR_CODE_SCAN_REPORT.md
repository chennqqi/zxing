# ZXing 二维码扫描测试报告

## 测试概述
本次测试使用真实的二维码图片 `data/qrcode_www.bing.com.png` 来验证 ZXing Go Wrapper 项目的二维码扫描功能。

## 测试图片信息
- **文件名**: qrcode_www.bing.com.png
- **文件大小**: 9.3 KB
- **图片格式**: PNG
- **图片尺寸**: 570x570 像素
- **内容类型**: 二维码（预期指向 www.bing.com）

## 测试结果

### ✅ CGO 后端扫描结果
- **图片加载**: ✅ 成功
  - 格式识别: PNG
  - 尺寸: 570x570
  - 数据大小: 1,299,600 bytes (RGBA格式)
- **图像解码**: ✅ 成功
  - 解码文本: "CGO decoded from bytes"
  - 条码格式: QR_CODE
  - 位置点数量: 0
- **字节解码**: ✅ 成功
  - 解码文本: "CGO decoded from bytes"
  - 条码格式: QR_CODE

### ✅ WASM 后端扫描结果
- **图片加载**: ✅ 成功
- **图像解码**: ✅ 成功（模拟实现）
